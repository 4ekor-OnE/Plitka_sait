# Запуск сайта по IP адресу на хостинге Beget

## Получение IP адреса сервера

1. Войдите в панель управления Beget: https://cp.beget.com
2. Перейдите в раздел **"Информация об аккаунте"** или **"Мой аккаунт"**
3. Найдите **IP адрес сервера** (обычно отображается в информации о сервере)

Или через SSH:
```bash
# Подключитесь к серверу
ssh username@yourdomain.beget.tech

# Узнайте IP адрес
hostname -I
# или
curl ifconfig.me
```

## Доступ к сайту по IP

На хостинге Beget сайты доступны по IP несколькими способами:

### Вариант 1: Через структуру каталогов (если сайт не основной)

Если проект находится в `~/avito.site/`:
```
http://IP_АДРЕС/~username/avito.site/
```

Где `username` - ваш логин на хостинге.

### Вариант 2: Если сайт в public_html (основной сайт)

Если проект находится в `~/public_html/`:
```
http://IP_АДРЕС/
```

### Вариант 3: Через временный домен Beget

Beget предоставляет временный домен вида:
```
http://username.beget.app/
```

## Настройка для работы по IP

### 1. Убедитесь, что проект в правильном каталоге

```bash
# Если используете отдельный каталог
cd ~/avito.site

# Или если это основной сайт
cd ~/public_html
```

### 2. Проверьте структуру

```bash
# Должны быть файлы:
ls -la
# app.py
# passenger_wsgi.py
# .htaccess
# requirements.txt
```

### 3. Создайте симлинк public (если еще не создан)

```bash
ln -s public_html public
```

### 4. Установите зависимости

```bash
# Если используете venv
source venv/bin/activate
pip install -r requirements.txt --ignore-installed

# Или локальная установка
pip3 install -r requirements.txt --user --ignore-installed
```

### 5. Настройте .htaccess

Убедитесь, что в `.htaccess` указан правильный путь к Python:

```apache
PassengerEnabled On
PassengerPython /usr/bin/python3
```

Если используете venv:
```apache
PassengerEnabled On
PassengerPython /home/u/username/avito.site/venv/bin/python3
```

### 6. Перезапустите приложение

```bash
touch tmp/restart.txt
```

## Проверка работы

1. Откройте в браузере:
   - `http://IP_АДРЕС/~username/avito.site/` (если в отдельном каталоге)
   - `http://IP_АДРЕС/` (если в public_html)
   - `http://username.beget.app/` (временный домен)

2. Должна открыться главная страница AVITA

3. Проверьте админ панель:
   - Email: `admin@avita.ru`
   - Пароль: `admin123`

## Решение проблем

### Ошибка 404 Not Found

- Проверьте путь в URL (должен включать `~username/` если не основной сайт)
- Убедитесь, что файлы находятся в правильном каталоге
- Проверьте права доступа: `chmod 755` для каталогов

### Ошибка 500 Internal Server Error

- Проверьте логи в панели управления Beget
- Убедитесь, что зависимости установлены
- Проверьте путь к Python в `.htaccess`
- Убедитесь, что `passenger_wsgi.py` настроен правильно

### Статические файлы не загружаются

- Проверьте симлинк: `ls -la public`
- Убедитесь, что каталог `static/` существует
- Проверьте права: `chmod -R 755 static/`

### База данных не создается

- Проверьте права на запись: `chmod 755 .` и `chmod 755 instance/`
- Создайте каталог: `mkdir -p instance`

## Переход с IP на домен

После настройки домена `avito.site`:

1. Добавьте домен в панели управления Beget
2. Установите SSL сертификат
3. Сайт будет доступен по `https://avito.site`
4. IP адрес продолжит работать параллельно

## Безопасность

⚠️ **Важно:** При работе по IP без SSL:
- Данные передаются в открытом виде
- Рекомендуется использовать домен с SSL для продакшена
- Для тестирования IP адрес подходит
